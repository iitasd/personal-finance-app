# PF Manager
A personal finance manager application

## Getting Started

After cloning, run following commands:

```
$ mvn clean install
```

Then copy 'personal-finance-app.war' file to '<TOMCAT_HOME>/webapps/' folder.

Start the server,

```
$ sh catalina.sh run
```
For debug (Port is 8000),
```
sh catalina.sh jpda run
```
